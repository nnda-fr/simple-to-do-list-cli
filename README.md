# Simple To-Do List CLI Application

Aplikasi To-Do List berbasis CLI sederhana yang dibuat menggunakan Python. Aplikasi ini memungkinkan pengguna untuk mencatat, melihat, menandai, dan menghapus tugas harian dengan penyimpanan otomatis ke file teks (`tugas.txt`).

## 🚀 Fitur
- **Tampilkan Tugas:** Melihat daftar tugas beserta statusnya.
- **Tambah Tugas:** Memasukkan tugas baru ke dalam daftar.
- **Tandai Selesai:** Mengubah status tugas menjadi selesai (✔).
- **Hapus Tugas:** Menghapus tugas tertentu dari daftar.
